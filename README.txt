프로그램을 실행하게 되면,

Enter mod number:

라고 출력됩니다. 이때, hashing 할 때 mod operand를 입력해 주시면 됩니다.
그 이후엔,

Enter buffer number:

라고 출력됩니다. 생성하시고 싶은 buffer의 갯수를 1이상 99이하의 정수로 입력해주시면 됩니다.
그러고 나면 hashQueue가 생성되며 hashQueue와 함께 freeList도 함께 출력이 됩니다.

Enter a block number which you want use (-1 to quit) :

가 출력되고 난 후 getblk 알고리즘을 통해 얻고 싶은 block의 blockNum을 입력해 주시면
해당되는 시나리오와 함께 얻고자 하는 block을 얻었음을 확인할 수 있습니다. -1을 입력해 주시면 프로그램을 종료합니다.
